# 面试训练助手插件使用手册

## 安装
1. 执行打包命令生成独立版 `.vsix` 文件（如 `interview-trainer-standalone.vsix`）。
2. 在 VS Code/Windsurf 中执行 `Extensions: Install from VSIX...` 安装。
3. 独立版无需安装 Continue 扩展，可单独运行。

## 更新插件
1. 拉取最新代码后，在 `interview-trainer/` 目录执行 `npm install`。
2. 打包生成新 VSIX：`npm run package`（产物在 `interview-trainer/build/interview-trainer.vsix`）。
3. 安装/覆盖安装：
   - VS Code：`code --install-extension <path-to-vsix> --force`
   - Windsurf：`windsurf --install-extension <path-to-vsix> --force`
   - 或在 IDE 中继续使用 `Install from VSIX...` 覆盖安装。

## 兼容性
- 基于 VS Code Extension API 开发，理论上可运行于大多数 VS Code 内核 IDE（Windsurf、Cursor、VSCodium 等）。
- 若 IDE 支持 `Install from VSIX...` 或提供 CLI 安装命令，可直接安装 `interview-trainer.vsix`。
- 若视图不显示，请确认 IDE 已开启 Webview、侧边栏视图能力，并支持 `vscode.open` 等基础命令。

## 基本使用
1. 打开侧边栏「面试训练助手」。
2. 录音或导入音频后点击「开始分析」。
3. 在结果标签页查看转录文本、声学分析、面试评价。
4. 点击「保存结果」可打开生成的 Markdown 报告。

## 常用按钮
- 开始录音/停止录音：麦克风录音，结束后自动生成音频数据。
- 导入音频：支持常见音频格式（内部会转成 16k PCM）。
- 开始分析：触发 ASR、笔记检索、评价与报告写入。
- 设置：打开 `api_config.yaml` 配置文件。
- 历史记录：查看过往记录并打开报告。

## 结果位置
默认输出到工作区 `sessions/YYYYMMDD/<topic-slug>/`：
- `attempt-XX-*.wav`：录音文件
- `*.md`：报告
- `attempts.json`、`meta.json`：结构化记录

## 常见问题
1. 提示缺少 API Key：请在配置文件中填写百度 ASR 与千帆模型密钥。
2. 网络失败：检查网络、代理或超时设置。
3. 录音失败：确认系统麦克风权限开启。
