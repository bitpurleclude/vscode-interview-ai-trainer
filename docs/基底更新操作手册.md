# Continue 基底更新操作手册

本仓库通过 `git subtree` 引入 Continue，更新时无需改动插件代码。

## 添加上游远程（已执行）
```bash
git remote add continue-upstream https://github.com/continuedev/continue.git
```

## 同步 Continue 最新版本
```bash
git fetch continue-upstream main
git subtree pull --prefix continue continue-upstream/main --squash
```

## 冲突处理
1. 冲突只会出现在 `continue/` 子树内。
2. 自定义代码位于 `continue/extensions/vscode/src/interviewTrainer/`，不会与上游冲突。
3. 解决冲突后重新构建并运行测试。
