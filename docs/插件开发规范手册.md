# 面试训练助手插件开发规范手册

## 代码结构
- 后端逻辑：`continue/extensions/vscode/src/interviewTrainer/`
- 前端 UI：`continue/gui/src/pages/interviewTrainer/`
- 配置模板：`continue/extensions/vscode/src/extension/config/interview_trainer/`
- 静态资源：`continue/extensions/vscode/src/extension/assets/interview_trainer/`

## 命名与隔离
- 新增模块统一使用 `it_` 前缀。
- 不修改 Continue 原有功能的业务逻辑；新增功能以新文件形式插入。

## 关键入口
- 视图提供者：`InterviewTrainerWebviewViewProvider`
- 插件入口：`InterviewTrainerExtension`
- 主流程：`it_runAnalysis`

## 配置策略
- 默认模板来自扩展资源目录。
- 实际可编辑配置存放在 `globalStorage/interview_trainer/`。
- API Key 建议通过 VS Code SecretStorage 注入。

## 开发流程建议
1. 新增能力前先补充协议类型（`core/protocol/interviewTrainer.ts`）。
2. 在 `InterviewTrainerExtension` 中注册新消息处理。
3. UI 页面通过 `IdeMessenger` 发起请求。
